# BoloNet

BoloNet is a custom operating system built from scratch written in **Lua** for **CC: Tweaked**, running inside **Minecraft**.

## Features

- Graphical desktop environment
- Multi-user accounts
- Persistent user data
- Per-user desktop layouts
- Modular application system

## Getting Started

1. Install BoloNet.
2. Launch the system.
3. Create your first account.
4. The first account automatically becomes the administrator.

> Runtime database files are generated automatically and are not included in this repository.

## Technologies

- Lua 5.2
- CC: Tweaked
- Minecraft
- CraftOS-PC
